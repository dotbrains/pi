# pi

This setup is opinionated. It:

- sets up GitHub dark default as the theme
- adds Firecrawl tools for searching, scraping, and crawling
- updates the bottom bar with useful status information
- adds subagents to Pi
- adds workflows to Pi
- adds an ask-user tool for multiple-choice questions

Setup instructions are available in `./SETUP.md`.
